Install WordPress & Xampp di Windwos

Setup Perangkat Lunak

Download & Install Xampp dan Wordpres

Xampp: https://www.apachefriends.org/download.html Wordpress: https://wordpress.org/download/

extract file wordpress ke folder xampp/htdocs

Jalankan xampp 'As Administrator' dan start service Apache & MysQl

Lanjut dengan membuat database di phpmyadmin, buka browser dan ketikan localhost/phpmyadmin. Setelah laman muncul, klik menu Database lallu isikan nama database dan klik Create.

Lanjut buka wordpress di browser dengan mengetikan localhost/wordpress
